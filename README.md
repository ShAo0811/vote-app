# vote-app

## 项目介绍
这是一个基于React和Vite构建的投票应用程序。

## 技术架构
- React 19
- Vite 6
- 现代JavaScript (ES6+)

## 安装教程
1. 克隆仓库
   ```
   git clone https://gitee.com/zhen-you-gao-tou-ba/vote-app.git
   ```
2. 安装依赖
   ```
   npm install
   ```
3. 启动开发服务器
   ```
   npm run dev
   ```

## 使用说明
1. 开发环境下，访问 http://localhost:5173 (或终端显示的端口)
2. 生产环境构建: `npm run build`
3. 预览生产构建: `npm run preview`
